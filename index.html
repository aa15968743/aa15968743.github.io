Hello ~以下是一些常用的Linux命令行命令： 文件和目录操作 - ls：用于列出目录内容。例如 ls -l 可以以长格式显示文件的详细信息，包括权限、所有者、大小、修改时间等。 - cd：用于切换目录。如 cd /home/user 会将当前目录切换到 /home/user 。 - pwd：显示当前工作目录的绝对路径。 - mkdir：创建新目录。例如 mkdir new_folder 会在当前目录下创建名为 new_folder 的新目录。 - rmdir：删除空目录。若目录非空，要先用 rm 命令删除文件后才能删除目录。 - cp：复制文件或目录。如 cp file1.txt /home/user/ 会将 file1.txt 复制到 /home/user/ 目录下。 - mv：移动或重命名文件或目录。例如 mv oldname.txt newname.txt 是重命名， mv file.txt /new/dir/ 是移动文件。 - rm：删除文件或目录。使用 rm -r 可以删除目录及其内容，要谨慎使用，因为删除后很难恢复。 文件查看和编辑 - cat：用于查看文件内容，将文件内容输出到终端。例如 cat file.txt 。 - less：和 cat 类似，但适合查看大型文件，可上下翻页浏览。 - nano、vim：文本编辑器。 nano 比较简单易用， vim 功能强大但学习曲线较陡。 系统信息查看 - uname：显示系统信息，如 uname -a 会输出内核名称、主机名等多种系统相关信息。 - top：动态查看系统资源使用情况，包括CPU、内存等的占用率和进程信息。 - ps：查看进程状态，如 ps -ef 可以显示所有进程的详细信息。 网络操作 - ping：用于测试主机之间的连通性，如 ping google.com 可以测试与谷歌服务器是否能连通。 - ifconfig（在有些系统中被 ip addr 替代）：查看和配置网络接口信息。
